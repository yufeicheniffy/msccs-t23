Help me!

Help me! is a tool to help public safety officers in retrieving information from Twitter during an emergency. This tool collects data from Twitter and runs a classifier to retrieve valuable knowledge from this data. Classified data is visualised to enable the user to get information he or she wants. Help me! involves two simple steps:

1
Use the searchbar to search for an existing emergency. This will start the tool retrieving tweets for us. For example: "forest fire California" will make a call to the Twitter API to search for relevant tweets.
2
Filter the returned tweets by the provide filters. Available filters are: type of incident, priority, order (e.g. chronological) and media.
